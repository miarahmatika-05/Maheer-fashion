## Tentang Proyek

**Maheer Fashion POS** adalah aplikasi web modern yang menggantikan pencatatan manual (buku/Excel) untuk mengelola transaksi dan inventaris toko fashion. Sistem mendukung:

- Penjualan **omni-channel** (Offline, Shopee, TikTok Shop) dalam satu database terpusat.
- Tracking stok berdasarkan kombinasi **variasi ukuran (S/M/L/XL) × warna**.
- Dashboard analitik **real-time** untuk pengambilan keputusan bisnis.
- Dua level akses: **Admin** (pemilik/manajer) dan **Kasir** (staf).

## Local Setup

### Prasyarat

Pastikan tools berikut sudah terinstal di mesin Anda:

- **Node.js** v18.17.0 atau lebih baru
- **npm** v9+ atau **yarn** v1.22+ atau **pnpm** v8+
- **Git** v2.30+
- Akun [Supabase](https://supabase.com) (gratis)

### 1. Clone Repository

```bash
git clone https://github.com/your-org/maheer-fashion-pos.git
cd maheer-fashion-pos
```

### 2. Install Dependencies

```bash
npm install
# atau
yarn install
# atau
pnpm install
```

### 3. Setup Supabase

1. Buat proyek baru di [app.supabase.com](https://app.supabase.com).
2. Masuk ke menu **SQL Editor** dan jalankan script migrasi di folder `supabase/migrations/` secara berurutan.
3. Aktifkan **Row Level Security (RLS)** untuk semua tabel sensitif (lihat `supabase/policies/`).
4. Salin **Project URL** dan **anon public key** dari menu **Settings → API**.

### 4. Konfigurasi Environment Variables

Duplikat file `.env.example` menjadi `.env.local`:

```bash
cp .env.example .env.local
```

Isi nilai yang diperlukan (lihat bagian [Environment Variables](#environment-variables)).

### 5. Jalankan Development Server

```bash
npm run dev
```

Aplikasi berjalan di `http://localhost:3000`.

## Environment Variables

| Variabel | Keterangan | Wajib |
|---|---|---|
| `NEXT_PUBLIC_SUPABASE_URL` | URL proyek Supabase Anda | ✅ |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Anon/public key Supabase | ✅ |
| `SUPABASE_SERVICE_ROLE_KEY` | Service role key (hanya server-side) | ✅ |
| `NEXTAUTH_SECRET` | Secret untuk enkripsi sesi | ✅ |
| `NEXT_PUBLIC_APP_URL` | URL aplikasi (contoh: `http://localhost:3000`) | ✅ |
| `LOW_STOCK_THRESHOLD` | Ambang batas stok rendah (default: `5`) | ❌ |

> ⚠️ **Jangan pernah commit file `.env.local` ke repository.** File ini sudah dimasukkan ke `.gitignore`.

## CHANGELOG

### [1.0.0] — 2026-05-02

**Initial Release — PRD v1.0.0**

#### Added
- Modul Autentikasi: Login/logout via Supabase Auth, manajemen sesi JWT.
- Modul Dashboard Analitik: Grafik revenue/profit, pie chart channel, bar chart ukuran produk, KPI cards.
- Modul POS: Katalog produk dengan pencarian SKU, keranjang transaksi, pilihan channel (Offline/Shopee/TikTok Shop), konfirmasi transaksi atomik.

- Deployment pipeline ke Vercel.
